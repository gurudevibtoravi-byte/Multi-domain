## 🔌 API Endpoints

1. `GET /api/tickets` - Get all tickets
2. `POST /api/tickets` - Create ticket
3. `PUT /api/tickets/:id` - Update ticket
4. `DELETE /api/tickets/:id` - Delete ticket
5. `GET /api/stats` - Get statistics
6. `GET /api/metadata` - Get form options
