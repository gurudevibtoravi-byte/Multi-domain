# Quick Start Guide - 30 Seconds

## Step 1: Install Dependencies
```bash
cd "c:\Users\User\OneDrive\Desktop\HTML"
npm install
```

Wait for completion...

## Step 2: Start Server
```bash
npm start
```

You should see:
```
✅ Multi-Domain Support Triage Server running on http://localhost:3000
📊 Dashboard: http://localhost:3000
```

## Step 3: Open Browser
Go to: **http://localhost:3000**

## Step 4: Test It
1. Fill the form
2. Click "Submit Ticket"
3. See ticket appear instantly
4. Click ticket to view details
5. Update status and delete as needed

## ✅ Done!

Your system is fully functional!

## 🛑 Stop Server
Press `Ctrl + C` in terminal
